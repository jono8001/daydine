Market readiness.

A read-only admin view of configured DayDine markets. It now loads generated QA/readiness outputs so you can see whether each market is ready, warning, or blocked before selling, publishing, or expanding.

Current operating model: new towns are still added through repo configuration and GitHub Actions. This page does not yet trigger pipelines; it displays generated QA status from assets/market-readiness/*.json.
